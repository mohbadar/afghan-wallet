### Report Utility Library of Anar Framework

#### Usage Guide

```
		<dependency>
	               <groupId>af.gov.anar.lib</groupId>
	                <artifactId>anar-lib-reportutil</artifactId>
                        <version>${project.version}</version>
		</dependency>

```

#### Features

- Report Entity Annotations
- Annotation Parser Configs
