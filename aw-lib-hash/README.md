### Hashing Utility Library for Anar Framework

#### Usage Guide

```
		<dependency>
	               <groupId>af.gov.anar.lib.hash</groupId>
	                <artifactId>anar-lib-hash</artifactId>
                        <version>${project.version}</version>
		</dependency>

```
